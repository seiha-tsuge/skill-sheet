### QRCode Hangtag

#### 2020/05〜2020/06

#### プロジェクト概要

商品を購入した一般消費者が商品購入後、下げ札のQRコードをスマホで読み取ることで、ブラウザベースで商品に紐付くコンテンツ(機能説明)を閲覧可能にする。

#### 担当業務

- 画面/API/DB設計、API/フロントエンド開発、単体テストを担当
- Lambdaを用いたアプリケーション開発に従事
- AWS Amplify + Vue.jsを用いたフロントエンド開発に従事

バックエンドは、TDDおよびクリーンアーキテクチャを採用することで品質管理を徹底。AWSは初めて触れる技術でしたが、日々の開発を通じてスキルを徹底してキャッチアップ  
年始に受注予定案件を前倒しで開発に着手するも、同時に複数案件を受注できたため、一旦プロジェクトを中止

#### 使用技術

AWS(Lambda, API Gateway, CloudFront, S3, Cognito, Amplify), TypeSctipt, Vue.js, Vuetify, Python, SQLAlchemy, MySQL
