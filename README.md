### 可視化システム(Mobile Data Viewer)

#### 2019/03〜2019/05

#### プロジェクト概要

このプロジェクトは、MaaS（Mobility as a Service）として、都市部の渋滞や移動困難者、公共交通のドライバー不足などの社会問題を解決することを目的としています。

#### 担当業務

APIおよびフロントエンドの開発、そして単体テストを実施しました。バックエンド開発では、Flaskを使用して、ユーザー管理機能用のAPIを開発。フロントエンド開発では、ゼンリンAPIや内製API(携帯基地局情報や自動車の走行データ)を利用して、これらの情報を地図上に視覚的に表示する機能を組み込みました。この機能によって、ユーザーは地図上で様々なデータを確認できるようになりました。

#### 取り組んだこと

このプロジェクトでは、1人称で機能開発を担当しました。これは、独力で開発を担当することを意味し、技術力だけでなく、タスク管理の能力も重要でした。開発にあたっては、しっかりと計画を立て、時間管理を徹底し、コードの品質にも細心の注意を払いました。TypeScriptとFlaskに関しては、プロジェクト開始時には熟知していませんでしたが、進行中にこれらを学び、使いこなす必要がありました。オンライン教材や書籍を活用し、実際にコードを書きながら習得しました。加えて、経験豊富なエンジニアからのコードレビューも受け、そのフィードバックを取り入れることで、技術力を高めました。私が開発した機能は、プロジェクトの要求を満たし、期待通りに機能しました。新しい技術の習得により、私のスキルセットは大きく広がり、将来のプロジェクトにも役立つようになりました。ベテランエンジニアからの学びは、今後のキャリアにとっても貴重な経験となりました。

#### 使用技術

AWS(RDS, EC2, S3), TypeScript, Vue.js, Vuetify, Python, Flask, Flask-RESTful, SQLAlchemy, MySQL
